# Common Patterns (Template)

Use this template for new deployment capability specs.

## Capability

Single-sentence statement of contract.

## Inputs

- IaC/helmfile/config files involved
- Environment variables and required defaults

## Expected Behavior

- Ordered rules and dependency constraints
- Cloud-specific branching behavior where relevant

## Scenarios

1. Scenario name
   - Given
   - When
   - Then
   - Verify: command(s) and file link(s)

## Non-Goals

- Explicitly state out-of-scope behavior.
